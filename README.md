ATM Management System :-
A modern, responsive **ATM Management System UI** built using **HTML, CSS, and JavaScript**.

Live Page :-  https://ayushipatidar93028-cyber.github.io/ATM_Management-System/

Features
- Card Insert & PIN Authentication
- Check Account Balance
- Withdraw Cash (Quick Amount Options)
- Deposit Cash
- Change ATM PIN
- Mini Statement (Transaction History)
- Professional ATM UI Design
- Fully Frontend (No Backend)

Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Google Fonts (Inter)

Screens
- Welcome Screen
- Login (PIN Entry)
- Dashboard
- Withdraw / Deposit
- Mini Statement
